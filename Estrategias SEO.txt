
1. Nombres de paginas

A cada una de las paginas, salvo Index, le corregí el nombre, sacandole los números y dejandole un nombre mas comprensible, corrigiendo dentro del html todos los enlaces.

2. Títulos

Siguiendo ese criterio, coloqué también los títulos adecuados dentro de la metaetiqueta correspondiente

3. Keywords

Para todas las paginas mantuve las mismas keywords, ya que el sitio completo esta diseñado para el mismo efecto

4. Description

En cada pagina agregué la descripción adecuada, que explica en forma breve lo que puede encontrarse en cada una,
respetando la recomendación de que la extensión sea menor de 150 chars.

En particular en index se realizó la descripción que debería salir publicada en una búsqueda

5. Encabezados h1
Se ajustaron los encabezados de cada página a h1 para ser identificados 

6. Tamaño de fotografías

Se ajustaron todas las fotos para mantener una buena velocidad de carga

7. Pagina de error 404

Se creó una página de error para mantenerse dentro del site

8. Site Map

Se creo un sitemap para encontrar fácilmente todas las secciones

9. Se diseño el sitio completo en forma responsive pensando en mobile first

10. Se revisaron todos los textos buscando que sean claros, concretos y libres de faltas de ortografía

11. Se agregará un sitemap xml una vez que el sitio esté online y acesible desde XML Sitemaps
